AVATAX ADAPTOR for C ++ integrations

This is a console app that runs a simple GetTaxRequest to the account specified via the usage:

CPP_GetTax.exe URL accountnumber licensekey companycode

Example 1:

CPP_GetTax.exe https://development.avalara.net 1100000000 A1B2C3D4E5F6G7H8 DEFAULT

Example 2:

CPP_GetTax.exe https://development.avalara.net user.name@company.com yourpassword DEFAULT
